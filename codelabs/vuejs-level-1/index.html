Use Vue.js as a "simple external library" for exercises 1 to 8:

This series of exercises is aimed at validating Vue.js learning objectives.

Note: Don't waste time on the graphical part. (A library will be presented to achieve a more "credible" design) But if you already know bootstrap feel free to use it.

You will build a small website which lists products and has a shopping cart functionality.

You can use any html editor, or use https://codesandbox.io/ with the Static HTML Template (do not use Vue.js template as it uses SFC by default).

Examples are provided throughout the exercises to help you, but you are not required to arrive exactly at the same result.

Create an HTML page displaying a list of products.

Each product has:

  • a reference number (id),
  • a title
  • a price
  • some products display the indication "unavailable".

A few products are enough for this preparation step.

Example

Example with Bootstrap (optional)

Add Vue.js to your page.

Define the products in a data structure, which Vue.js will display when the page loads.

This should replace the created list from (exercise 1).

Note: Visually there will be no difference, but all the product data should now be in the javascript part and not the html part of the document.

Warning: Check the documentation for using Vue with CDN .

Either use Global Build or ES Module Build, but pay close attention to the script tag (with or without module)!
  • Add the necessary html fields above the list to enter a new product (title, price, ...).
  • Add an "Add" button.
  • When the button is clicked, the product is added to the products.
  • The input fields are reset.

Example

Example with Bootstrap (optional)
  • Add a text field that filters the displayed product list based on the input made (only products whose name contains the input are displayed). Bonus: make it Case-insensitivity.
  • Add an "Available products only" checkbox. If the box is checked, only available products are displayed.
  • Add a dropdown list "Sort by" with the choices "Name", and "Price".

Example

Example with Bootstrap (optional)

Add a "Add to cart" functionality as follows:

  • when a displayed product is clicked, it is added to a "cart". The quantity of each product corresponds to the number of clicks. Bonus: the button should not be enabled on unavailable products.
  • Display the "cart" in the page. Display the name, quantity, price and price times quantity
  • Display a line with the total number of items in the cart and the total price.

Example

Example with Bootstrap (optional)
  • Add for each product in the cart the ability to modify the product quantity. Bonus: do not allow to go below 1.
  • Allow to remove a product from the cart.
